# True 3D Subway Surfers (React + Three.js)

This is a **true 3D** version of Subway Surfers built with:

- **React Three Fiber** (Three.js + React)
- **@react-three/drei** for helpers
- **Procedural infinite track**
- **Real 3D collision**
- **Smooth camera follow**

## Best GitHub References Used for Accuracy

1. **DanielLin0516/SUBWAY-SURFERS** ← Best modern Three.js implementation
   - https://github.com/DanielLin0516/SUBWAY-SURFERS
   - Live: https://subway-surfers-threejs.vercel.app/

2. **KSVSC/Subway-Surfers** (Classic WebGL + Three.js)
   - Police dog, jetpack, realistic obstacles

3. **swetanjal/WebGL-Subway-Surfer**
   - Very detailed features (flying boots, moving trains, inspector)

## How to Run

```bash
npm run dev
```

Then open the game (3D version will be the default).

## Controls
- **← →** or **A D** : Switch lanes
- **SPACE** or **↑** : Jump
- **↓** : Slide
- **P** : Pause
